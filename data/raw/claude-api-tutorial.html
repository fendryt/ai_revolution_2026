What is an API?

Think of an API as a conversation. You ask Claude a question, and Claude responds. But instead of typing in a chat interface, your code sends the question and receives the answer.

üí° Real-world analogy It's like sending a text message to a really smart friend. You write your message, hit send, and wait for their reply. The Claude API works the same way!

The basic flow is simple:

You: "Hey Claude, explain quantum physics simply" Claude: "Quantum physics is the study of how tiny particles..."

Building Your First Request

Every API call to Claude needs three essential ingredients. Think of it like baking a cake - miss one ingredient and it won't work!

const response = await fetch("https://api.anthropic.com/v1/messages", { method: "POST", headers: { "Content-Type": "application/json", }, body: JSON.stringify({ model: "claude-sonnet-4-20250514", max_tokens: 1000, messages: [ { role: "user", content: "Your question here" } ] }) });
üîç Breaking down each part ‚ñº

model: Which version of Claude to use (like choosing which expert to talk to)

max_tokens: Maximum length of the response (roughly 4 characters per token)

messages: Your conversation history - at minimum, your question

Reading Claude's Response

When Claude responds, you get back a JSON object with several pieces of information. The most important part is the actual text response.

const data = await response.json(); // The response structure looks like this: { "content": [ { "type": "text", "text": "This is Claude's actual response!" } ], "model": "claude-sonnet-4-20250514", "role": "assistant" } // To get just the text: const message = data.content[0].text;

Let's parse a response together. Here's a sample response from Claude:

5

Having a Conversation

The real power comes from multi-turn conversations. You can keep the context by including previous messages in your requests.

messages: [ { "role": "user", "content": "What's the capital of France?" }, { "role": "assistant", "content": "The capital of France is Paris." }, { "role": "user", "content": "What's famous there?" } ]

Notice how the second question references "there" - Claude knows we're talking about Paris because of the conversation history!
