## Under The Hood

### Prompt utilized (29JAN2026):

 |`Create an interactive dashboard that tells a story with data about AI adoption, usage patterns, or impact.`
 
**Development Journey:**
- Claude ([Chat](https://claude.ai/new), Sonnet 4.5)
  - [Attempt-1](https://claude.ai/share/3cfcb753-6a68-488f-b72e-5805f51071f0) ***The Let's See...*** 
    - "[The AI Revolution](https://fendryt.github.io/pre.anthropic.io/data/final/20260129_ClaudeSonnet45_ai-adoption-dashboard_update01.html)" Dashboard
    - TARGET: Request Claude to draw real-world data to generate a quick report to develop an interactive dashboard.
    - DISCOVERY: Good and fast response. Elegantly simple but displayed professional ***static*** dashboard. Insufficient data points, lack of richness to develop a progressive story to showcase AI maturity (adoption, usage, impact) comparison. 
- Claude Desktop ("Cowork", Opus 4.5)
    - Attempt-2 ***The Show & Impress Me***
      - "[AI Adoption Intelligence](https://fendryt.github.io/pre.anthropic.io/data/final/20260129_ClaudeOpus45__ai-adoption-dashboard_original.html)" Dashboard
      - TARGET: Expect Claude to take time to draw real-world data, and to generate a more sophisicated report to develop an interactive dashboard.
      - DISCOVERY: Significant improvement over "Sonnet 4.5". Excellent data points and professional interactive dashboard. The **eureka** moment of "non-deterministic" experience: Same prompt generated two different data points. *What if I push the same prompt on different AI service providers with different user accounts?*
    - Attempt-3 ***Finale: The Boss Dashboard*** ("Cowork", Opus 4.5)
      - "[The State of AI Revolution](https://fendryt.github.io/pre.anthropic.io/data/final/20260129_ClaudeOpus45__ai-adoption-dashboard_original.html)" Dashboard
      - TARGET: Utilized real-world data points drawn by different popular AI service providers with different user accounts to compile and integrate all the resources to generate the multi-view interactive dashboard. Screenshot { [1](images/Anthropic_Claude_Cowork01.png),  [2](images/Anthropic_Claude_Cowork02.png) }
      - Utilized Google [Gemini 3](https://gemini.google.com/app) with three user accounts to generate four { [1](/data/final/20260129_Gemini3Personal01_ai_interactive_dashboard_update07.html),  [2](/data/final/20260129_Gemini3Personal02_ai_interactive_dashboard_original.html),  [3](/data/final/20260129_Gemini3Biz_ai_dashboard_update01.html),  [4](/data/final/20260129_Gemini3Biz_ai_dashboard_update02.html) } interactive dashboards with real-world data. 
      - Utilized Microsoft [Copilot GPT-5.1](https://copilot.microsoft.com/) with two user accounts but only the licensed M365 was able to generate interactive [dashboard](/data/final/20260129_CoPilotBizGPT51_AI_Interactive_DashBoard_update03.html) with real-world data.
      - OpenAI (Free) [ChatGPT](https://chatgpt.com/) generated interactive [dashboard](/data/final/20260129_OpenAIChatGPT_ai_adoption_dashboard.html) with real-world data.
